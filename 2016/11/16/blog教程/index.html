git相关指南

安装git

https://git-scm.com/

安装node.js

https://nodejs.org/en/

安装Hexo

https:/Hexo.io/
npm install hexo-cli -g

  1. 进入E盘: E: 新建文件夹 hexo
  2. 进入hexo: cd hexo
  3. 新建文件夹,命令： hexo init blog
  4. 进入新建文件夹： cd blog
  5. npm install
  6. hexo server
  7. 登陆 localhost4000

    github 执行指南

  8. 进入 E:
  9. 新建文件夹 编辑代码
  10. cd ~/E/文件夹
  11. 进入github官网 复制仓库地址
  12. git clone 仓库地址
  13. 进入仓库 ： cd <仓库>
  14. git status
  15. git add .
  16. git status
  17. 提交： git commit -m “”
  18. git pull origin master
  19. git push origin master

    博客主题下载

  20. 进入hexo.io 点击 Themes 进入主题 点击喜欢的主题名
  21. 进入对方github 下载主题 完成修改名称 只保留主题名
  22. 用编辑器打开blog里面的_config.yml文件 把theme 改成 主题名
  23. 命令: hexo server

    上传域名

  24. 进入github 新建仓库 .github.io
  25. 复制 仓库地址
  26. 在E盘 里新建一个文件夹 github
  27. 命令： E:
  28. cd github
  29. git clone <仓库地址>
  30. 退出 github文件夹 cd ..
  31. 进入 hexo : cd hexo
  32. 进入 blog: cd blog
  33. 命令： hexo generate 完成后会生成一个public 文件
  34. 把public里面的所有内容复制到github文件夹里面的仓库文件夹里
  35. 进入仓库域名文件夹
  36. 后执行github执行指南
  37. 如果执行命令 git commit -m “” 后出现提示：please tall me your email 执行命令： git config –global user.email”邮箱”
  38. 如果提示： Did you mean this config . 先执行 git config –global user.name”用户名”
