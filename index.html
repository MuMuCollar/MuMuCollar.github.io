git相关指南

安装git

https://git-scm.com/

安装node.js

https://nodejs.org/en/

安装Hexo

https:/Hexo.io/
npm install hexo-cli -g

  15. git add .
  16. git status
  17. 提交： git commit -m “”

  19. git push origin master

all&none&any

这是我在codewars里面做的每日一题，记录着我从一个最低级慢慢升到高级的过程，从中可以看到很多大牛的优秀demo，我从中获得了宝贵的经验，受益匪浅。

quection:

As a part of this Kata, you need to create three functions that one needs to be able to call upon an array:

作为这一型的一部分，你需要创建三个函数，一个需要能够召唤一个数组：

all

This function returns true only if the predicate supplied returns true for all the items in the array

此函数只有在数组中的所有项目提供的回报率为真时才返回真值

Examples:

[1, 2, 3].all(isGreaterThanZero) => true

[-1, 0, 2].all(isGreaterThanZero) => false

none

This function returns true only if the predicate supplied returns false for all the items in the array

Examples:

[-1, 2, 3].none(isLessThanZero) => false

[-1, -2, -3].none(isGreaterThanZero) => true

any

This function returns true if at least one of the items in the array returns true for the predicate supplied

Examples:

[-1, 2, 3].any(isGreaterThanZero) => true

[-1, -2, -3].any(isGreaterThanZero) => false

思路：

any: 循环数组，都满足条件则为true，否则为false；
none：循环数组，都不满足条件则为true，否则为false；
any: 循环数组，有一个满足条件则为true，都不满足条件则为false；

固定函数：

function isGreaterThanZero (num) {
return num > 0;
}
function isLessThanZero (num) {
return num < 0;
}

我的代码：

Array.prototype.all = function (p) {
for(var i =0; i<this.length; i++){
if(!p(this[i])){
return false;
}
}
return true;
};

Array.prototype.none = function (p) {
for(var i =0; i<this.length; i++){
if(p(this[i])){
return false;
}
}
return true;
};

Array.prototype.any = function (p) {
for(var i =0; i<this.length; i++){
if(p(this[i])){
return true;
}
}
return false;
};

大神代码：

Array.prototype.all = function (p) {
return this.filter(p).length == this.length;
};

Array.prototype.none = function (p) {
return this.filter(p).length == 0;
};

Array.prototype.any = function (p) {
return this.filter(p).length > 0;

};

精髓：

filter: 返回满足条件的数组，不是返回布尔值；

first

这是我在codewars里面做的每日一题，记录着我从一个最低级慢慢升到高级的过程，从中可以看到很多大牛的优秀demo，我从中获得了宝贵的经验，受益匪浅。

Quick Start

quection:

实现叠重功能

Examples:

accum(“abcd”); // “A-Bb-Ccc-Dddd”

accum(“RqaEzty”); // “R-Qq-Aaa-Eeee-Zzzzz-Tttttt-Yyyyyyy”

accum(“cwAt”); // “C-Ww-Aaa-Tttt”

思路：

  1. 字符串大写
  2. 建立新的数组， 遍历字符串，每个字符加入数组 ， 每一个大写字母后面的小写字母的个数 等于这个大写字母在字符串中的位置
  3. 把数组转化成字符串，用“-”链接；

我的代码：

function accum(s) {         
    var result = [];   
    var str = s.toUpperCase();
    for(var j =0,le = str.length; j<le ;j++){
        var st = str[j];
        var f = j;
        while(f--){
             st += str[j].toLowerCase();
        }
        result.push(st);  
    }
    return result.join('-')
}

大神代码：

function accum(s) {
  return s.split('').map((x,index) => x.toUpperCase()+Array(index+1).join(x.toLowerCase())).join('-');
}

积累经验：

map: 1. 传入的函数要有返回值

  1. map 方法返回的是一个新数组。

=> 箭头函数 继承上面的this 前面要加（）
